Crystals Collector!

The goal of this game is to match the target score shown on the screen. Each crystal shown below holds a unique point value. Click on a crystal to see the point value added to your score. Match the score exactly and you win! If you go over the target score, you lose.

The target score and the crystals' point values will be randomly generated at the start of each game. Good luck!
